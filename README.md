#  PetNesto

PetNesto is a responsive pet products e-commerce website designed to simplify online shopping for pet owners. The platform offers an intuitive interface for browsing pet products, searching items, and exploring products based on pet type and category.

## Features

- Responsive user interface
- Product categorization
- Search functionality
- Modern shopping experience
- Clean navigation
- Mobile-friendly design

## Tech Stack

- React
- JavaScript
- HTML5
- CSS3

## Installation

```bash
cd PetNesto
npm install
npm start
```
